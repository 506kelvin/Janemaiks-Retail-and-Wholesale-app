# JaneMaiks Retail Assistant 

AI-powered Android retail assistant for small businesses and family-owned shops.
Built to help shop attendants quickly manage inventory, calculate prices, record sales, and interact with a smart AI assistant — all from a mobile phone.

---

#  Features

##  AI Shop Assistant

The built-in AI assistant helps attendants interact with the shop naturally.

Examples:

* “How much is white handkerchief?”
* “What sales have been made today?”
* “How many Kasuku biros remain?”
* “Suggest a selling price for cooking oil”
* “What items are low on stock?”

The assistant supports:

* fuzzy product matching
* typo tolerance
* conversational follow-ups
* inventory lookups
* sales summaries
* pricing suggestions

#  Requested Items Tracker

Track products customers ask for but are unavailable.

Example:
Customer requested:

```text

```

Available:

```text
fx82MS
```

The app stores:

* requested item name
* request frequency
* request date
* notes

Useful for restocking decisions.
